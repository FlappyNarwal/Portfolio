Web Design

Samen met mijn team heb ik voor een opdrachtgever een Datavisualisatie gemaakt die inzicht zou moeten geven op Etnisch profileren door de politie binnen Amsterdam.

Tijdens deze opdracht vervulde ik de rol van Visual Designer en heb ik gewerkt aan een responsive design. Mijn teamgenoten waren front-end developers en hebben de website uiteindelijk gebouwd. Deze kan je bezoeken door op de onderstaande knop te drukken.
